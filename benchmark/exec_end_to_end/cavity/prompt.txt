Set up and run a 2D lid-driven cavity flow simulation in OpenFOAM
using icoFoam.

Domain:
- Unit square, 0.1 m x 0.1 m x 0.01 m (single cell in z).
- Structured hex mesh with 20 x 20 x 1 cells generated by blockMesh.
- scale factor 0.1 in blockMeshDict.

Boundary conditions on velocity U:
- movingWall (top wall)  : fixedValue uniform (1 0 0)  m/s
- fixedWalls (other walls): noSlip
- frontAndBack           : empty

Boundary conditions on pressure p:
- movingWall  : zeroGradient
- fixedWalls  : zeroGradient
- frontAndBack: empty

Fluid: Newtonian, kinematic viscosity nu = 0.01 m^2/s.

Solver / time control:
- Solver         : icoFoam
- startTime      : 0
- endTime        : 0.5 s
- deltaT         : 0.005 s
- writeInterval  : 20 (write every 0.1 s)

Numerical schemes and linear solvers:
- Time integration     : Euler
- Divergence           : Gauss linear
- Laplacian            : Gauss linear corrected
- Pressure solver      : PCG with DIC preconditioner, tol 1e-6, relTol 0.05
- Velocity solver      : smoothSolver symGaussSeidel, tol 1e-5, relTol 0
- PISO                 : 2 correctors, 0 non-orthogonal correctors

Task:
Produce a complete OpenFOAM case directory (0/, constant/, system/) that
matches the parameters above. Run blockMesh followed by icoFoam and confirm
the case reaches endTime without a FOAM FATAL exit.
